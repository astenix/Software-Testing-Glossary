\section{Decision Table Testing}
\label{sec:Decision Table Testing}

\rindex{\textbf{D}!Decision Table Testing}An excellent logical tool to capture certain kinds of system requirements and to document internal system design. 

They are used to record complex business rules that a system must implement. 

They can serve as a guide to creating test cases. Decision tables are a vital tool in the tester's personal toolbox.

Not every analyst, designer, programmer and/or tester are familiar with this technique\footnote{~Quote from \russianq{A Practitioner's Guide to Software Test Design} by \rindex{\textbf{L}!Lee Copeland}Lee Copeland, 2004, Artech House, Inc.}.

Sad but true.
